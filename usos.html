usos

Inspirado na página uses.tech, segue uma lista de ferramentas que eu uso no meu dia-a-dia.

Visual Studio Code

Visual Studio Code é um editor de código-fonte leve, mas poderoso, que é executado em sua área de trabalho e está disponível para Windows, macOS e Linux. Ele vem com suporte integrado para JavaScript, TypeScript e Node.js e tem um rico ecossistema de extensões para outras línguas e tempos de execução (como C++, C#, Java, Python, PHP, Go, .NET).

Extensões do VS Code
  • Prettier

    Prettier é um código opinativo para assuntos. Ele impõe um estilo consistente analisando seu código e remimindo-o com suas próprias regras que levam em conta o comprimento máximo da linha, embrulhando código quando necessário.

  • Live Server

    Inicie um servidor de desenvolvimento local com recurso de recarga ao vivo para páginas estáticas e dinâmicas.

  • GitHub Copilot

    O GitHub Copilot é uma ferramenta de inteligência artificial desenvolvida pelo GitHub em conjunto com a OpenAI, afim de auxiliar usuários de ambientes de desenvolvimento integrados como Visual Studio Code, Visual Studio, Neovim e JetBrains por meio do sugestão de código para preenchimento automático.

  • Eva Theme

    Eva Theme for VSCode é um tema de código de coloração semântico que o faz mais confortável na programação, já suporta esses idiomas. São 6 estilos no total: Dark, Dark Bold, Dark Italic; Leve, Leve Negrito, Itálico Leve. Os estilos Bold tornam as palavras-chave e os nomes de função mais ousados. Os estilos itálicos fazem palavras-chave itálicos.

Visual Studio

Microsoft Visual Studio é um ambiente de desenvolvimento integrado da Microsoft para desenvolvimento de software especialmente dedicado ao .NET Framework e às linguagens Visual Basic, C, C++, C# e F#.

Everything

Everything é um utilitário de pesquisa na área de trabalho para Windows que pode localizar arquivos e pastas rapidamente pelo nome. É lançado sob uma licença de software livre que permite modificação e redistribuição comercial, exigindo apenas atribuição.

ShareX

O ShareX é uma ferramenta de captura de tela com uma impressionante variedade de ferramentas de edição de vídeo e imagem. Depois que um arquivo é salvo, o software oferece uma variedade de filtros, opções de edição e ferramentas de upload para compartilhar rapidamente qualquer quantidade de arquivos.

RaiDrive

O RaiDrive é um software para Windows que sincroniza serviços de hospedagem na nuvem com o computador do usuário. A partir de uma janela simples de configuração, o programa cria drives virtuais no sistema que simulam discos rígidos de verdade, para que seja possível acessar e editar os arquivos guardados online sem precisar estar conectado ou abrir o navegador.
